-------------------------------------------------------------
Gowalla dataset for location-based recommendation experiments
-------------------------------------------------------------

This dataset was gathered by collecting public data from the Gowalla service in October 2011. It contains 30.367 georeferenced spots in new York city, and information about 357.753 visits for 19.183 users of Gowalla.

The following files are included in the download:

* README.txt : The document you are reading now.

* spots.txt : A tab-separated file with spots in New York, containg ids, names, and geospatial coordinates.

* users.txt : A tab-separated file with Gowalla users, containg ids, names, hometowns and geospatial coordinates.

* highlights.txt : A tab-separated file with spots marked as highlights by Gowalla users, containg spot and user ids, together with a textual description with the type of spot.

* users-spots.txt : A tab-separated file with spots visited by Gowalla users, containg spot and user ids, together with a Boolean value indicating if the user frequently visits the spot.
